Introdução a React

React é uma biblioteca, uma framework para JavaScript, usado para desenvolver UI - interfaces de Usuários - em aplicativos web

A documentação pode ser encontrada em react.dev.

Instalando o React

Instalando através do Vite.

npm create vite@latest digite o nome do projeto Escolher a opção 'React' e depois 'JavaScript' cd 'nome-do-projeto' npm install

A estrutura criada é a seguinte:

index.html: O ponto de entrada da aplicação. Aestrutura HTML principal do projeto. Aqui temos uma div com o ID root, onde nosso React vai injetar os componentes.
src/main.jsx: O arquivo principal que inicializa o React. Iimporta o componente App e renderiza dentro da div com ID root.
src/App.jsx: O componente principal da aplicação.
