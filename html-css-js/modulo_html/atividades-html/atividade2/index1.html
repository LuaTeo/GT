Tags iniciais:

A Tag "h" representa título. Começando sempre do título que quer dar maior importância até o de menor importâcia. Conceitualmente existe seis níveis de títulos (h1, h2, h3, h4, h5 e h6).

A partir de então, temos "p" que representa tag de parágrafo.

Em seguida, as tag de escrita mais simples: "b" que serve para destacar texto em negrito; "i" que serve para destacar texto em itálico; "u" que serve para destacar texto sublinhando.

Tags de listas:

As listas são divididas em ordenadas, representadas pela tag "ol", e não ordenas, representadas pela tag "ul". Onde a diferença está na forma em que são apresentadas, enquanto as ordenas são numeradas as não ordenadas apresentam pontos no lugar dos números.
Dentro delas ficam os itens de listas descritos pela tag "li".

Síntaxe x Semântica:

HTML trata da síntaxe e da semêntica da escrita do tento. A síntaxe se refere a forma como deve ser escrito para que seja criado e a semântica se refere ao sentido do texto através do uso das tags.
Exemplo:
A tag "b" é a forma (a síntaxe) com que é possível criar um texto em negrito. Já a tag "strong" também deixa um texto em negrito, no entanto ela dá relevância a quele texto para os buscadores.

Outras formatações

Outra formatação interessante está relacionada a tag "code", que é usada para escrever um texto em formato de código em linguagem de programação.
Exemplo:

                        autor = "J.R.R. Tolkien"
                        ano_publicacao = 1954

                        # Exibindo as informações do livro
                        print("Informações do Livro:")

                        print(f"Número de Páginas: {numero_paginas}")
                        print(f"Editora: {editora}")
