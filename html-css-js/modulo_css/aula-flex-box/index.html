Aula FlexBox

Melhorando o Layout

O conceito de flexbox vem da utilização da propriedade display:flex dentro de um seletor, que pode ser chamado de container. A partir disso todos os elementos dentro do container serão flexíveis, isso quer dizer que poderão ser modificados mais facilmente.
