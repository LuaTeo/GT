Uso de Box Models

O conceito de box model se refere a estilização em blocos. Os elementos de HTML são descritos em "caixas". Na folha de estilo css "style.css" é definido uma div contendo propriedades como: width, height, content, padding, margin, border e etc.

Então, todo conteúdo dentro da caixa estará sujeito as propriedades dela.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Eaque, ducimus porro amet illo nisi deleniti maiores asperiores! Numquam, deserunt perferendis illo tempora pariatur aliquam culpa ab soluta voluptate deleniti esse.

Uso da propriedade "Display"

A propriedade display é muito utilizada para alterar a forma com que as caixas se alinham.

display:none = Não renderiza a caixa e possibilitando que o mesmo não ocupe espaço de tela.
display:inline = Não permite configurar estilo e formatação em caixa. Tendo o mesmo comportamento em linha.
display:inline-block = Possue os dois comportamento, inline e block, mantendo as caixas lado a lado.

Propriedade "Visibility"

A propriedade visibility pode mostrar ou esconder um elemento sem afetar o layout da página.
Também pode ocultar linhas ou colunas em uma tabela.

Uso da propriedade Position

Essa propriedade permite posicionar os elementos HTML onde quiser.

Positio:static = É a posição padrão do HTML. E se organiza de acordo com a tag utilizada. Também pode ser posicionado em top, left, botton, right e etc.
Position:absolute = Define a posição absoluta da div em relação ao elemento pai principal. Ou seja, a posição é relacionada a posição do documento.
Position:relative = Define a posição em relação a div (ou elemento) pai "associado".
Position:fixed = Usa-se essa propriedade quando tiver um elemento dentro de um container pai e quiser posicionar fora, no documento. Também pode ser posicionado em top, left, botton, right e etc.
