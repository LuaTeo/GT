A hierarquia da folha css é dado de cima para baixo, no geral, daí o nome "cascading style sheet". No entanto, a maior hierarquia acontece do id, depois class, depois o nome da tag.

Se a página de estilo está com a mesma tag estiliza, então será lido do style mais inferior.

Caso esteja iniciando com uma class e o mais inferior for uma tag. Então o que sera estilizado (ou adotado) é a class.

A flag !mportant pode ser aplicada a cada elemento de estilo para dá uma hierarquia maior a estilização.
